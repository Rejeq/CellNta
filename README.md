# CellNta

N-Dimensional cellular automata

## How to build
```shell
git clone https://github.com/Rejeq/CellNta
cd CellNta
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
cmake --build .
```
